Proyecto Wall-E

Descripción del Proyecto

Este proyecto utiliza tarjetas Raspberry Pi equipadas con sensores de humedad y temperatura para monitorear el estado de una planta. El sistema es capaz de regar la planta o ajustar la temperatura según las necesidades detectadas.

Componentes del Sistema

Sensor de Humedad

Monitorea la humedad del suelo y activa el riego automático cuando es necesario.

Sensor de Temperatura

Detecta la temperatura ambiente y ajusta el entorno para optimizar el crecimiento de la planta.
